# Resources

A curated reading list — books, papers, courses, blog posts, and code worth your time.
To add a link: drop a bullet under the matching `## Category` (or create a new one), then run `npm run build:content`.

## Operating Systems
- [xv6 Book (RISC-V)](https://pdos.csail.mit.edu/6.828/2023/xv6/book-riscv-rev3.pdf) — MIT's teaching OS, the canonical way to learn how a Unix kernel actually works.
- [xv6 Annotated](https://github.com/palladian1/xv6-annotated) — Line-by-line commentary on the xv6 source for debugging and reading.
- [xv6 Source (public)](https://github.com/mit-pdos/xv6-public) — The original xv6 codebase to read alongside the book.

## Databases
- [B-Trees and Database Indexes](https://planetscale.com/blog/btrees-and-database-indexes#btrees-in-mysql) — PlanetScale's clear explainer on B-trees in MySQL/InnoDB.
- [Postgres Internals Through Pictures](https://momjian.us/main/writings/pgsql/internalpics.pdf) — Bruce Momjian's classic visual tour of Postgres internals.
- [Build Your Own Database](https://cstack.github.io/db_tutorial/) — Write a sqlite-clone in C, one chapter at a time.
- [Postgres Planner Development & Debugging](https://dev.to/ashenblade/postgresql-planner-development-and-debugging-47mc) — Practical guide to debugging the Postgres query planner.
- [Understanding the Postgres Planner: Scan Paths vs Plans](https://www.highgo.ca/2024/03/22/understand-postgresqls-planner-simple-scan-paths-vs-plans/) — A deeper read on planner internals.
- [CMU 15-445 — Database Systems](https://15445.courses.cs.cmu.edu/fall2025/schedule.html) — The best free database course on the internet, by Andy Pavlo.

## Storage (SSD / HDD)
- [Coding for SSDs](https://codecapsule.com/2014/02/12/coding-for-ssds-part-1-introduction-and-table-of-contents/) — Multi-part series on how SSDs work and what that means for your code.
- [HDD vs SSD — Conceptual Overview](https://gemini.google.com/share/b873de81ca1f) — A quick refresher on the differences.
- [Understanding L2P Block Address Translation (LBA)](https://americas.kioxia.com/content/dam/kioxia/en-us/business/memory/mlc-nand/asset/KIOXIA_Managed_Flash_BOS_P5_Understanding_L2P_Block_Address_Translation_Tech_Brief.pdf) — Kioxia's brief on logical-to-physical address mapping inside flash.

## Hardware & CPU
- [CPU.land](https://cpu.land/) — How CPUs and operating systems collaborate, told as a journey.
- [Intel 4004 Datasheet](https://datasheets.chipdb.org/Intel/MCS-4/datashts/intel-4004.pdf) — The first commercial microprocessor's original datasheet.

## Networking & Web
- [Beej's Guide to Network Programming](https://beej.us/guide/bgnet/html/) — The friendliest introduction to writing C network code (build your own HTTP server).

## Distributed Systems
- [Netflix Graph Abstraction](https://www.infoq.com/news/2026/03/netflix-graph-abstraction/) — How Netflix unified entity relationships at scale.
- [CMU 15-440 — Distributed Systems](https://www.synergylabs.org/courses/15-440/index.html) — Course materials covering the foundations of distributed systems.
- [MIT 6.824 Lecture Notes](https://github.com/yyin-dev/MIT6.824) — Compiled notes from the famous distributed systems course.

## Computer Systems & Architecture
- [CMU 15-213 — Computer Systems (lectures)](https://www.youtube.com/playlist?list=PLtv4PQFH2yMas_SCMTEaE4eUIWWWiwISj) — The Bryant & O'Hallaron course in video form.
- [CMU 15-213 Labs](https://github.com/codeAligned/CMU-15213-Lab) — Lab solutions and reference for the same course.
- [CMU 15-213 — My Notes](https://github.com/LakshimiRam-073/Notes/tree/main?tab=readme-ov-file) — My own notes while working through 15-213.
- [Illinois CS341 Coursebook](https://cs341.cs.illinois.edu/coursebook/) — A genuinely good free systems coursebook.

## Build Your Own X
- [Build Your Own X](https://github.com/codecrafters-io/build-your-own-x?tab=readme-ov-file) — Tutorials for re-implementing classic technologies from scratch.
- [Internal Workings](https://github.com/LakshimiRam-073/Internal-workings) — My personal notes on how things work under the hood.

## Languages
- [Essential Go — Project-based Learning](https://github.com/abbasovdev/essential-go/blob/main/chapters/01_intro/03_how_to_use/README.md) — Learn Go by building real projects, chapter by chapter.
